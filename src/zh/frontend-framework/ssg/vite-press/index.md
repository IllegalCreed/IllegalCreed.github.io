# VitePress

这是一个适合展示编程技术类文档的静态网站生成框架。基础文档采用 `MD` 编写，支持使用 `Vue` 进行扩展。

## 评价

VitePress 是 VuePress 1 的精神继任者（spiritual successor），由 Vue 团队维护：

**优点**
- 基于 Vite，dev server 与构建速度极快
- Markdown 中可直接嵌入 Vue 组件
- 整体风格与交互更现代化
- 内置 Shiki 代码高亮、自带搜索、i18n、SSR 支持

**缺点**
- API 较精简，部分定制（自定义主题、复杂 layout）需要熟悉 Vue 3
- 仍是 ESM-only 项目，与某些 CJS 工具链集成略繁琐

属于目前最主流的开发者文档工具之一
