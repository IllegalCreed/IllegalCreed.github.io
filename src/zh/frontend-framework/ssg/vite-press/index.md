# VitePress

这是一个适合展示编程技术类文档的静态网站生成框架。基础文档采用 `MD` 编写，支持使用 `Vue` 进行扩展。

## 评价

该框架继承自 `VuePress` ：

**优点**
- 支持编写及展示 `Vue` 组件 
- 整体风格及交互更现代化
- 支持通过插件进行扩展

**缺点**
- 官方文档过于简介，很多要去 `VuePress` 官网上查看详情

属于目前市面上比较流行的文档展示工具
