# 浏览器基础

面向前端工程师的浏览器工作原理体系：从多进程架构与导航编排、渲染管线（解析→布局→绘制→合成），到浏览器侧的存储与多层缓存，最后落到浏览器执行环境自带的安全防护机制。与「计算机网络基础」章互补——网络章讲数据怎么在网络上传，本章讲浏览器拿到数据后怎么变成页面、存在哪、怎么防护。
