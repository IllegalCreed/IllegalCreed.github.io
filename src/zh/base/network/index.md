# 计算机网络基础

面向前端工程师的网络知识体系，自底向上覆盖协议栈：从分层模型、链路层与网络层，到传输层 TCP/UDP、应用层各协议（DNS / HTTP / HTTPS / 跨域 / 实时通信），再到接入与移动网络。重心放在前端高频实用核心（HTTP、HTTPS、跨域、缓存、DNS），同时保留较完整的网络体系。
