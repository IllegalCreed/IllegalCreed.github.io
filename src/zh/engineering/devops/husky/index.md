# Husky

超快的现代原生 `git` 钩子，在提交或推送时自动检查提交消息、代码并运行测试。

## 评价
因为我没有用过同类竞品，所以不好评价，只能通过我使用过程中的问题提出一些不足之处。
- 对于 windows 支持较差，会出现权限甚至文件编码问题
- 逻辑有些粗糙，比如初始化时，对于prepare脚本它采用直接覆盖的策略，而非合并或交给用户选择。
