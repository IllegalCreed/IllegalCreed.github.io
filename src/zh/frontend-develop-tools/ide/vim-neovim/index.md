# Vim/Neovim

经典的**模态文本编辑器**谱系：**Vim**（vi 后裔，终端无处不在）与 **Neovim**（Vim 的现代化 fork，Lua + 内置 LSP）。

## 评价

### 优点

- **模态编辑 + 组合式命令语言**（operator + motion + text object），纯键盘、效率极高
- 终端原生、无处不在，SSH / 远程开发必备；启动极快、资源极省
- 高度可定制（`.vimrc` / `init.lua`），插件生态成熟
- **Neovim 现代化**：Lua 一等公民、内置 LSP / Treesitter、异步；SO 2024 调查 admiration 最高（编辑器之首）

### 缺点

- 学习曲线陡峭（模态与命令语言需肌肉记忆）
- 开箱即用能力弱，需大量配置才达现代 IDE 体验
- 图形化调试、重型语言支持不如 GUI IDE
- 配置碎片化（发行版/插件众多，新手易迷失）
