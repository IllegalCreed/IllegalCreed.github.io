# Antigravity

Google 推出的 **agent-first 开发平台**（agentic IDE），由 Gemini 3 驱动；2025 年末新品。

## 评价

### 优点

- **agent-first 范式**：以 **Agent Manager** 为中心，异步编排多个 agent 并行工作，而非侧栏聊天机器人
- agent 自主 **plan → execute → verify**，可跨 **editor / terminal / browser** 端到端完成任务
- **Artifacts**（可审查交付物：实现计划 / 任务列表 / walkthrough / 截图 / 浏览器录制）取代原始日志
- 类 **Google Doc 的反馈**：在 artifacts 上高亮 + 评论，agent 实时纳入而不打断流程
- Gemini 3 Pro 主力，并支持 Claude Sonnet 4.5 / GPT-OSS；个人公测免费

### 缺点

- 2025-11 新品，稳定性与上下文记忆仍在打磨，版本/配额/定价快速变动
- 基于 VS Code fork（业界报道源自 Windsurf 代码库），生态成熟度不及老牌 IDE
- 浏览器验证依赖 **Google Chrome + 专用扩展**
- 公测期企业合规能力偏弱
