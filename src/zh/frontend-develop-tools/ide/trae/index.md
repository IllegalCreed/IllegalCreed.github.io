# Trae

字节跳动出品的 **AI 原生 IDE**，国内市场第一；提供「IDE」与「SOLO」双模式。

## 评价

### 优点

- **国内第一 AI IDE**（IDC Q1 2025 约 41.2% 市场份额、600 万+用户）
- **IDE / SOLO 双模式**：从开发者完全掌控，到 AI 主导（plan → 编码 → 测试 → preview → 部署）
- **SOLO Agent** 可编排多个自定义 agent 组成「AI 团队」；Plan / Spec 模式产出规划与规格文档
- **CUE** 智能补全；VS Code fork，可从 VS Code / Cursor 一键导入设置
- 国内版直供豆包等国产模型、符合合规；长期 beta 免费，性价比高

### 缺点

- 闭源 VS Code fork；部分平台特定扩展需手动下载 VSIX
- 国内版与国际版的模型/合规分化（如美国用户不可用 GPT、MiniMax 系列）
- 与字节生态（火山引擎/豆包）绑定较深
- 功能更名频繁（Builder → Agent、SOLO Coder → SOLO Agent）
