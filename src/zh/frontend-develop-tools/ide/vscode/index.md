# VSCode

微软开源的跨平台代码编辑器，凭借海量扩展生态成为当今最主流的开发工具。

## 评价

### 优点

- 连续多年稳居最受欢迎编辑器（Stack Overflow 2025 开发者调查约 75.9% 使用率），生态与社区规模无可匹敌
- 扩展市场（Marketplace）插件海量，几乎覆盖所有语言、框架与工作流
- 内置 Git 集成、调试、集成终端、远程开发，开箱即用
- 2026 起深度集成 AI（Chat / Agent 模式 / MCP），从「编辑器」演进为「agentic 开发平台」
- 免费开源（源码 MIT 协议），跨 Windows / macOS / Linux

### 缺点

- 基于 Electron，启动速度与内存占用高于原生编辑器（如 Zed、Sublime Text）
- 开箱的语义分析与重构能力弱于 JetBrains 系（WebStorm/IntelliJ），需靠扩展补齐
- 扩展质量参差，安装过多会明显拖慢性能
- 微软分发的 VS Code 默认采集遥测（telemetry），且 Marketplace 与若干扩展为微软专有（完全开源者可选 VSCodium）
