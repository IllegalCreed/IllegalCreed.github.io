# WebStorm

JetBrains 出品的 **JavaScript / TypeScript 专精 IDE**，以开箱即用的语义理解与重构著称。

## 评价

### 优点

- **开箱即语义化**：类型推断驱动的补全、跨文件 Find Usages、声明/实现跳转，无需配插件就「理解」整个 JS/TS 项目
- **重构是一等公民**：Rename / Extract / Inline / Change Signature 带预览 + 冲突检测，跨文件安全更新引用
- **重型工具全内置**：DataGrip 级数据库工具、HTTP Client、专业调试器开箱即用
- **JetBrains 独有安全网**：Local History（独立于 Git 的本地改动追踪）+ Shelve
- 2025.1 起内置 **AI Free** 层（无限补全 + 本地模型）+ agentic 助手 **Junie**

### 缺点

- 基于 JVM / IntelliJ 平台，**资源占用重**（启动慢、内存可达 GB 级）
- 商业订阅付费（非商业用途免费、学生与开源项目免费）
- 扩展生态规模小于 VS Code
- 概念与快捷键体系庞大，学习曲线较陡
