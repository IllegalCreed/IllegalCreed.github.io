# Sublime Text

**C++ 原生编写**的极速跨平台代码编辑器，以多光标编辑与 Goto Anything 著称。

## 评价

### 优点

- **C++ 原生、极快、低资源**，启动与大文件处理远胜 Electron 系编辑器
- **多光标/多选**编辑强大（Quick Add Next、列选择）
- **Goto Anything** 快速导航；ST4 支持 GPU 渲染（三端，最高 8K）
- 配置纯 JSON，Python 插件 API + Package Control 生态

### 缺点

- 商业授权（可**无限期免费评估**，但持续使用需购买，个人 $99 一次性含 3 年更新）
- **LSP / 语言能力非内置**，需装第三方 LSP 包
- 扩展生态远小于 VS Code；无内置 Git（需配 Sublime Merge）
- 开箱即用的现代 IDE 能力弱于 VS Code
