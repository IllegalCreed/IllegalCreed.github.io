# 前端可视化

把数据、三维场景、动画与交互以图形方式呈现的能力集合。按用途分为四类：

## 图表

基于 Canvas / SVG / WebGL 的数据图表库：ECharts、D3.js、Chart.js、Mermaid。

## 三维

WebGL 之上的 3D 渲染与引擎：

- [**Three.js**](./three/) —— 跨浏览器的通用 3D 库，是 3D 可视化的事实标准
- WebGL（底层 3D API）、Babylon.js（完整 3D 引擎）、CesiumJS（地理空间 3D）

## 动画

补间与运动库：GSAP、Framer Motion、Lottie。

## 拖拽

拖拽排序与交互：Sortable.js。
